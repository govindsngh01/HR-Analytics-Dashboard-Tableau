# HR-Analytics-Dashboard-Tableau
Interactive HR Analytics Dashboard built in Tableau to analyze employee attrition, job satisfaction, demographics, and workforce trends.
HR Analytics Dashboard 📊

Project Overview

This project presents an interactive HR Analytics Dashboard built using Tableau. The dashboard helps analyze employee attrition patterns, workforce demographics, and job satisfaction trends to support data-driven HR decisions.

Key Metrics

- Employee Count: 1,470
- Attrition Count: 237
- Attrition Rate: 16.12%
- Active Employees: 1,233
- Average Age: 37

Dashboard Features

- Department-wise Attrition Analysis
- Education Field-wise Attrition
- Employee Age Group Distribution
- Job Satisfaction Analysis
- Gender-wise Attrition Insights
- Interactive Filters and Visualizations

Tools Used

- Tableau Public
- Data Visualization
- Data Analytics

Key Insights

- Sales department shows the highest attrition.
- Most employees belong to the 26–35 age group.
- Attrition trends vary across education fields and gender groups.
- Job satisfaction levels provide useful insights into employee retention.
